# QQ 群聊

## 用户群

欢迎所有 Minecraft 玩家加入 HMCL 用户群，入群请遵守以下群规：

1. 禁止谈论政治相关、色情、违法犯罪、金钱往来等敏感内容，禁止辱骂、造谣诽谤、恶俗等行为，禁止讨论精神疾病、LGBT 等话题，请尊重每一个人；
2. 禁止刷屏、倒垃圾、引战、宣泄个人情绪、刻意误导等破坏讨论环境的行为；
3. 禁止上传 EXE 和 APK 文件；
4. 禁止使用第三方模块/插件在群聊中发送特殊消息；
5. 禁止讨论多人游戏中的作弊工具；
6. 禁止发布与 Minecraft 无关的广告，可以适当宣传服务器、整合包等相关内容，同一内容每人每周最多宣传一次；
7. 用户群交流主题为 Minecraft，请不要过多讨论其他游戏内容。

## 报错崩溃交流群

若你遇到了游戏非正常退出等问题，可以加入报错崩溃交流群寻求帮助。由于加群人数过多，我们会定期移出问题已被解决的用户。如在被移出后再次遇到新的问题，仍可重新申请加群。

<!--{% comment %}-->
> [!WARNING]
> <!--{% endcomment %}-->
> <!----{{'>'}} **警告** <br> <!---->
> 报错崩溃交流群仅处理游戏报错崩溃问题，无关话题请前往其它群聊讨论，入群请遵守以下群规：
<!----{{'>'}}
{: .notice--warning }
<!---->

1. 游戏非正常退出时，请点击左下角 <span class="btn btn--primary btn--small">导出游戏崩溃日志</span> 按钮（不是 <span class="btn btn--primary btn--small">日志</span> 按钮），并完整发送生成的报错压缩包
2. 遇到其他问题时，请详细说明情况并提供相关截图，以便我们提供帮助
3. 使用整合包时，请说明具体的整合包名称与版本
4. 请保持耐心，群管理和群友会协助解决问题。严禁辱骂、催促他人
5. 禁止讨论政治、色情、违法、金钱交易等敏感内容，禁止人身攻击、造谣诽谤等行为
6. 禁止刷屏、灌水、挑起争端、发泄情绪等影响群聊秩序的行为
7. 禁止使用第三方插件发送特殊消息；禁止讨论多人游戏作弊工具
8. 本群仅支持 HMCL 启动器相关问题。PCL 启动器用户请加入 978054335 群
9. 请勿重复发送文件，如需重新获取关注，请引用之前发送的消息
10. 因人数限制，已解决问题的用户会被移出群聊。如遇新问题可再次加入
11. 本群仅处理游戏崩溃相关问题，其他话题请到其他群讨论

欢迎加入报错崩溃交流群咨询解决方案：[Minecraft 报错崩溃交流群 (666546887)](https://qm.qq.com/q/nG0Ti1kJri)

## 灵车交流群

我们为 Linux、FreeBSD 等系统以及 ARM（不包括 macOS）、RISC-V、龙芯等设备的用户建立了一个独立的交流群。
如果你正在使用这些平台，欢迎加入 HMCL 灵车交流群：[HMCL 灵车交流群 (737461713)](https://qm.qq.com/q/C935haj8xW)。

本群不适合一般 Windows 和 macOS 用户加入。如果你不关心上面提到的这些平台，请移步[用户群](#用户群)。

## 公测群

如果你关心 HMCL 最新的开发进展，愿意积极测试 HMCL 的新功能并给出反馈，欢迎加入 HMCL 公测群：[HMCL 公测群 (954100320)](https://qm.qq.com/q/EO7Cx87Cz6)

本群用于讨论 HMCL 的开发和测试，禁止闲聊，并且随时会清理未积极参与反馈的用户。如果你想闲聊，请移步[用户群](#用户群)。
